# Dorker
## Installation Linux
```bash
$ sudo apt-get update
$ sudo apt-get install git
$ git clone https://github.com/Zexurei/Dorker/
$ cd Dorker
$ python2 dorkmaker.py
After Using Dork Use Dork Scanner
python2 dorkscanner.py
```
## Installation Termux
```bash
$ pkg update && pkg upgrade 
$ pkg install git 
$ pkg install python2 
$ pip2 install colorama
$ git clone https://github.com/Zexurei/Dorker/
$ cd Dorker
$ python2 dorkmaker.py
After Using Dork Use Dork Scanner
python2 dorkscanner.py
```
